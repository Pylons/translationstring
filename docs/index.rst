Internatl
=========

A library used by various `Repoze <http://repoze.org>`_ packages for
internationalization (i18n) duties.

This package provides a :term:`translation string` class, a
:term:`translation string factory` class, interpolation helpers, and
utilities that help Chameleon templates use translation facilities of
this package.
